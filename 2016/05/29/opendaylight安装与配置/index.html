opendaylight安装与配置
参考网址：https://wiki.opendaylight.org/view/Main_Page

直接下载编译后程序

li版本的opendaylight控制器的端口为6653，不是6633.

1、从https://www.opendaylight.org/downloads下载Lithium-SR3版本软件和Getting Started Guide文件。
2、上传到10.20.31.207服务器/home/ubuntu/downloads目录
3、按照下载的说明文件，解压运行karaf。

安装odl-l2switch-switch-ui插件
  1. cd /home/ubuntu/downloads/distribution-karaf-0.3.3-Lithium-SR3
  2. ubuntu@odl207:~/downloads/distribution-karaf-0.3.3-Lithium-SR3$./bin/karaf
  3. opendaylight-user@root>feature:install odl-l2switch-switch-ui
安装过程中出现错误，错误信息没能记下来，意思是要求java版本1.7以上，而服务器上版本为1.6。
退出karaf，安装java1.7
  1. opendaylight-user@root>logout
  2. ubuntu@odl207:~/downloads/distribution-karaf-0.3.3-Lithium-SR3$sudo apt-get install openjdk-7-jdk
安装完成后，其实还是使用的旧版本java，需要在/home/ubuntu/.bashrc中修改JAVA_HOME值，然后关闭终端，重新登录，发现java版本已经是新版本了。
再次进入karaf，安装odl-l2switch-switch-ui插件成功。

安装the DLUX web interface
根据说明文档中网址：https://wiki.opendaylight.org/view/OpenDaylight_DLUX:DLUX_Karaf_Feature安装
  1. opendaylight-user@root>feature:install odl-restconf odl-l2switch-switch odl-mdsal-apidocs odl-dlux-core
用浏览器可以打开 http://10.20.31.207:8181/index.html网址，能看到与控制器连接的of交换器流表已知的拓扑结构。
